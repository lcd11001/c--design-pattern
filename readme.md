# Udemy Course
- [Design Patterns in C# and .NET][2]
# install .net 6.0
- [Download link][1]
- Check install successful
```sh
    dotnet --version
```
# Create Console App
```sh
    dotnet new console -o HelloWorld --use-program-main
```
# Run the app
- Debug mode
```sh
    dotnet run
```
- Release mode
```sh
    dotnet run -c release
```
# Build the app
- Debug mode
```sh
    dotnet build
```
- Release mode
 ```sh
    dotnet build -c release
 ```
